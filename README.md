# message-notification-demo
> 消息通知：业务系统动态（周报、 审批、公告、VOC等 ）通过钉钉工作通知/群消息通知进行有效提醒
> 
> 需要登录[开发者后台](https://open-dev.dingtalk.com/#/)，在创建的应用中添加“chat相关接口的管理权限”
